How to run the DTH MANAGEMENT SYSTEM
1. Download the zip file

2. Extract the file and copy DTH MANAGEMENT SYSTEM folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name DTH

6. Import DTH.sql file(given inside the zip package in SQL file folder)

7.Run the script http://localhost/DTH
